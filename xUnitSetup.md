# xUnit Setup

1. Open the Solution Explorer.
2. Click on New Project...
3. Search for and select xUnit Test Project.
4. Run dotnet test to check that everything is working.
5. Right click on the test project directory and select Add Project Reference.
6. Select the project to which a reference should be added.
